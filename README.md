# Recipe Agent using Google ADK

## Overview

Recipe Agent is an AI-powered cooking assistant built using Google Agent Development Kit (ADK) and Gemini 2.5 Flash.

The agent helps users discover suitable recipes based on the ingredients they have available. It analyzes ingredients, recommends a recipe, estimates the meal cost, and provides easy-to-follow cooking instructions.

The project demonstrates how AI agents can combine large language model reasoning with custom tools to solve everyday problems.

---

## Features

* Accepts ingredients in natural language.
* Recommends recipes based on available ingredients.
* Categorizes ingredients into food groups.
* Estimates ingredient cost.
* Generates step-by-step cooking instructions.
* Uses Gemini 2.5 Flash for reasoning and decision-making.
* Integrates custom tools through Google ADK.
